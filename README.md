# 🗺️ Roadmap Dev 2026

Roadmap completo de desenvolvimento de software para as trilhas de **Frontend**, **Backend** e **Fullstack**, com foco no ecossistema **React / React Native / Node.js / TypeScript**.

## 📌 Sobre

Este documento é um guia de rota para formação de desenvolvedores, indo do absoluto básico até nível avançado/profissional. Cobre desde HTML semântico até cloud, IA e arquitetura de sistemas escaláveis.

### Trilhas cobertas

- **Frontend** — HTML, CSS, JavaScript, TypeScript, React, Next.js, React Native
- **Backend** — Node.js, APIs REST, PostgreSQL, Prisma, Docker, DevOps
- **Fullstack** — Tudo das trilhas acima + Cloud (AWS), IA (OpenAI, LangChain), IoT e Produto

## 🎯 Objetivo pessoal

Estou usando este roadmap como guia de estudos pessoal para aprender, na prática, as tecnologias listadas em cada fase — seguindo a ordem proposta e construindo projetos reais ao longo do caminho.

## 🛠️ Tecnologias do roadmap

`HTML` `CSS` `JavaScript` `TypeScript` `React` `Next.js` `React Native` `Expo` `Node.js` `Express` `Fastify` `PostgreSQL` `Prisma` `Redis` `Docker` `AWS` `Socket.IO` `Jest` `TailwindCSS` `OpenAI` `LangChain`

## 📄 Como usar

Abra o arquivo `index.html` no navegador ou acesse o deploy para visualizar o roadmap completo com todas as fases organizadas por nível e tempo estimado.
